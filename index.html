Projects

NLP-Subreddit Classification

Scenario-A university is organizing a panel discussion with experts in Science and Engineering. There are going to be live round of questions from in-person as well as remote audience during the panel discussion. There is going to be electronic entry of queries from remote audience as well as optionally from in-person audience. Aim: The aim of this project is to create a model that can classify the electronically entered questions between the two categories (Science and Engineering) for the moderator so that the questions can be directed to appropriate experts for answering/discussion. This is accomplished by training a model on the text gathered from two subreddits-AskScience and AskEngineers. The project involves data acquisition using reddit API (webscraping), exploratory data analysis, and training several classification models to find the best one.

Diabetic Retinopethy-Image Classification

Diabetic Retinopathy (DR) is a complication resulting from diabetes that affects the eyes. Currently DR is manually detected by trained clinians by examining the photographs of the retina and detecting the presence of lesions associated with the abnorminal blood vessels or haemorrage, that may represent as tiny red spots in the image. This manual detection is a very time consuming process. This project aims to create a model for automated detection (prediction) of the presence or absence of the condition by analyzing color fundus images, and if possible the severity of the condition. The project involves data pre-processing, training several models (including custom CNN, transfer learning etc.) with different strategies to overcome the imabalanced classes and analyzing the model performance to find the best one.
